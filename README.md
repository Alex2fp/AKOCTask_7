# AKOCTask_7

## Описание программы

`copy32` — утилита на C++ (в стиле C), которая копирует содержимое одного файла в другой, используя только системные вызовы `open`, `read`, `write`, `close`, `fstat` и `fchmod` для работы с файлами. Для буферизации применяется кольцевой буфер фиксированного размера 32 байта, который последовательно используется и при чтении, и при записи.

Программа сохраняет права доступа исходного файла. Благодаря этому исполняемые бинарные файлы и скрипты после копирования остаются исполняемыми, а обычные текстовые файлы не получают дополнительных прав.

## Использование

```bash
g++ -std=c++17 -Wall -Wextra -pedantic copy32.cpp -o copy32
./copy32 <исходный_файл> <файл_назначения>
```
